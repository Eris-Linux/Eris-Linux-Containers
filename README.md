# Eris Linux Containers

## Description

This directory contains the script needed to prepare containers for the Eris Linux system (see Eris Linux documentation at [www.eris-linux.net](https://www.eris-linux.net) ).

It also includes examples of containers that use the resources provided by Eris Linux (either as a standalone operating system or by using the Eris Linux API).

These examples are designed to be viewed, compiled, and installed one after another to cover most of the situations you will encounter while developing your application code.
